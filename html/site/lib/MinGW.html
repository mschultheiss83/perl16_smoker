NAME

MinGW - Minimalist GNU for Windows

DESCRIPTION

The MinGW PPM package installs the MinGW GCC compiler and make utility into the Perl tree at site/lib/auto/MinGW. It also creates links for the following commands in th site/bin directory so that they can be found on the PATH:

    ar
    dlltool
    g++
    g77
    gcc
    mingw32-make
    windres

The MinGW PPM package also requests installation of dmake as a prerequisite. ActivePerl will automatically use MinGW and dmake to build additional packages using the standard mechanism:

    perl Makefile.PL
    dmake
    dmake test
    dmake install

or using the CPAN shell:

    cpan Module::Name
